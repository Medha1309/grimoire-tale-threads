## Troubleshooting

### "Still can't connect"
1. Make sure you're using: `http://localhost:5173/` (not 3000 or 5174)
2. Try: `http://127.0.0.1:5173/`
3. Check if another app is using port 5173
4. Hard refresh: Ctrl+Shift+R

### "Page is blank"
1. Open browser console (F12)
2. Check for errors
3. Look at the Network tab
4. Try incognito/private mode

### "Image not showing on Contact page"
- This is normal! The gradient fallback is showing
- Add your image to see it
- The app works perfectly either way
