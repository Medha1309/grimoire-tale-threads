## ğŸ” For Authenticated Users

### Creating Content

#### New Post
1. Click **"New Post"** button (top right or bottom on mobile)
2. Fill in:
   - **Title** - Captivating discussion title (max 100 chars)
   - **Content** - Your thoughts, theories, questions (max 5000 chars)
   - **Tags** - Optional genre tags for categorization
3. Click **"Post"** to publish

#### Reply to Post
1. Scroll to bottom of post view
2. Type in the reply form
3. Click **"Reply"** to post

#### Reply to Reply (Nested)
1. Click **"Reply"** button on any reply
2. Form shows "Replying to [name]"
3. Type your response
4. Click **"Reply"** to post

#### Quote Reply
1. Click **"Quote"** button on any reply
2. Form auto-fills with quoted text
3. Add your response below the quote
4. Click **"Reply"** to post

## ğŸ› Troubleshooting

### "Please sign in" message
- You need to be logged in for this action
- Click "Sign Up" or "Login" in navbar

### Can't see Edit/Delete buttons
- These only appear on your own content
- Make sure you're logged in with the correct account

### Reply not posting
- Check minimum length (3 characters)
- Ensure you're logged in
- Check internet connection

### Like button not working
- Must be logged in to like
- Check if you already liked it
- Try refreshing the page

### Report not submitting
- Must be logged in to report
- Select a reason before submitting
- Check internet connection
