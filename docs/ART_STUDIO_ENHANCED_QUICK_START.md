## Advanced Features

### Blend Modes

- **Normal**: Standard drawing
- **Multiply**: Darkens colors
- **Screen**: Lightens colors
- **Overlay**: Combines multiply/screen
- **Difference**: Inverts colors
- **Lighten**: Only lighter pixels
- **Darken**: Only darker pixels

### Smoothing

- **0%**: Raw, unfiltered strokes
- **50%**: Balanced stabilization
- **100%**: Maximum smoothing

### Scatter

- **0%**: Precise placement
- **50%**: Moderate randomness
- **100%**: Maximum scatter

### Flow

- **1%**: Very light application
- **50%**: Medium coverage
- **100%**: Full opacity per stroke

### Symmetry Mode

- Mirrors all strokes across vertical center
- Perfect for symmetrical designs
- Toggle on/off anytime

### Grid Overlay

- 50px √ó 50px reference grid
- Helps with proportions
- Toggle on/off anytime

## Tips & Tricks

### For Best Results

1. **Use Smoothing**: Set to 50-70% for cleaner lines
2. **Layer Colors**: Use low opacity and build up
3. **Try Blend Modes**: Experiment with different modes
4. **Use Symmetry**: Great for faces and patterns
5. **Enable Grid**: Helps with perspective and alignment

### Brush Combinations

- **Blood + Multiply**: Deep, rich reds
- **Ethereal + Screen**: Bright glowing effects
- **Watercolor + Low Opacity**: Soft washes
- **Neon + Overlay**: Vibrant highlights
- **Smoke + Low Flow**: Atmospheric effects

### Workflow Tips

1. **Sketch First**: Use Charcoal with low opacity
2. **Add Color**: Switch to color brushes
3. **Add Details**: Use Ink for fine lines
4. **Add Effects**: Use Ethereal/Neon for glow
5. **Final Touches**: Add Blood/Decay for horror
