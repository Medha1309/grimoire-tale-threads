# Cursor System Summary

## Pages with Default Cursor
- **Landing Page** - Clean, professional default cursor
- **Login/Signup Pages** - Default cursor with atmospheric background effects
- **Library/Stories Pages** - Default cursor + enhanced torch effect that follows mouse

## Pages with Custom Cursors

### Forum - Gilded Ornament
- Luxurious golden rotating frame
- Ruby jewel center that pulses
- Sparkle particles when hovering
- Matches gilded parlour aesthetic

### Dollhouse/Diary - Vintage Key
- Pink vintage skeleton key
- Decorative ribbon that sways
- Floating hearts when hovering
- Vintage dust particles

### About - Mystical Ring
- Translucent purple ethereal ring
- Rotating with subtle glow
- Pulsing mystical dots
- Minimal particle effects

### Contact - Crosshair
- Translucent dark blue/grey design
- Clean geometric crosshair
- Subtle corner accents
- Gentle ink ripple when hovering

## Key Features
- All custom cursors use translucent fills (0.06-0.08 opacity)
- Smooth animations with Framer Motion
- Responsive to hover and click states
- Performance optimized
- No impact on accessibility
