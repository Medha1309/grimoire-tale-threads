## Recommended Settings

### For Sketching
- Brush: Charcoal
- Size: 5-10px
- Opacity: 50-70%
- Smoothing: 60%
- Scatter: 0%

### For Line Art
- Brush: Ink
- Size: 3-5px
- Opacity: 100%
- Smoothing: 70%
- Scatter: 0%

### For Painting
- Brush: Oil Paint
- Size: 15-30px
- Opacity: 80-100%
- Smoothing: 40%
- Scatter: 10%

### For Atmosphere
- Brush: Smoke/Watercolor
- Size: 30-50px
- Opacity: 20-40%

- Scatter: 30%

### For Effects

## Tips for Best Results

### Smoothing
- **Low (0-30%)**: Textured, rough
- **Medium (40-60%)**: Balanced
- **High (70-100%)**: Smooth, clean

### Scatter
- **None (0%)**: Precise
- **Low (10-30%)**: Subtle variation
- **High (50-100%)**: Painterly

### Flow
- **Low (1-30%)**: Transparent layers
- **Medium (40-70%)**: Build-up
- **High (80-100%)**: Solid coverage

### Blend Modes
- **Multiply**: Add shadows
- **Screen**: Add light
- **Overlay**: Enrich colors
- **Difference**: Special FX

## Performance Tips

### For Smooth Drawing
- Use moderate smoothing (50-70%)
- Keep scatter low (0-20%)
- Disable grid if not needed
- Disable symmetry if not needed

### For Large Artworks
- Work in sections
- Save frequently
- Use undo sparingly
- Clear history if slow

## Troubleshooting

| Issue | Solution |
|-------|----------|

| Strokes too rough | Increase smoothing |
| Colors too light | Increase flow/opacity |
