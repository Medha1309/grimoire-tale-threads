Quick Access

Pre-flight Checklist

  • Dev server is running (npm run dev)
  • You are logged in (required for desktop access)
  • Browser console is open (F12) to see errors
  • URL is http://localhost:5173/desktop

What Should You See?

  • Background: Teal gradient (#008080)
  • Desktop Icons: 8 icons in top-left (My Diary, Stories, Art Studio, etc.)
  • Taskbar: Gray bar at bottom with Start button and clock
  • Start Menu: Opens when you click Start button

Troubleshooting

If you see a blank page:

  • Check browser console (F12) for errors
  • Make sure you're logged in
  • Try refreshing the page

If you're redirected to login:

  • The desktop requires authentication
  • Log in first, then navigate to /desktop

If you see errors:

  • Check that react-draggable is installed
  • Run: npm install react-draggable --legacy-peer-deps
  • Restart the dev server
