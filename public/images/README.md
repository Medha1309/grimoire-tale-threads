# Images Folder

## Add Your AI Art Here

Place your AI-generated forest/moon image here as:
- `forest-moon.jpg` (recommended)
- Or any other name and update the path in `src/pages/Contact.tsx`

### Recommended Specs
- Format: JPG or PNG
- Size: 1920x1080 or higher
- File size: Under 500KB (compressed)

### If No Image
The app will show a beautiful gradient fallback background automatically.
