# 3D Models

## Skull Model Required

Place your skull GLB file here as `tarred_skull.glb`

You can download a free skull model from:
- Sketchfab (search for "skull" with downloadable/free license)
- TurboSquid free models
- CGTrader free section

The model should be:
- Format: GLB (GLTF binary)
- Filename: `tarred_skull.glb`
- Recommended poly count: 10k-50k triangles
- Centered at origin

Once you add the file, the About page will display the 3D possessed skull with:
- Scroll-driven twitches
- Idle drift animation
- Wet glint effects
- Possession spikes
